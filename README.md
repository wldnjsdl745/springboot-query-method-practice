# springboot-query-method-practice
Spring Boot, JPA, MySQL을 사용해 **Query Method**를 연습한 프로젝트

## 학습 내용
- Spring Data JPA Query Method
- 엔티티와 리포지토리 구성
- MySQL 연동
- 조건 기반 데이터 조회

## 기술 스택
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Gradle
- Lombok

## 핵심 개념
- Query Method는 메서드 이름만으로 조회 쿼리를 생성할 수 있는 기능입니다.
- 간단한 조건 검색을 빠르게 구현할 수 있습니다.
- 복잡한 조회는 JPQL, QueryDSL 등과 비교해서 학습할 수 있습니다.
